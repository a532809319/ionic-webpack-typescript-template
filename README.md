## How to start?

#### Install global dependencies

```
$ npm install cordova ionic webpack bower -g
```

#### Install local dependencies

Install npm dependencies

```
$ npm install
```

Install bower dependencies

```
$ bower install
```

Restore ionic project

```
$ ionic state reset
```

## How to work with?


Start webpack with watcher flag

```
$ webpack --watch
```

Serve ionic

```
$ ionic serve
```
